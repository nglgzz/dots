# [Dots](../README.md) – Fedora

**Requirements**

- Do a fresh install of Fedora.

- Run the following commands:

  ```bash
  git clone https://github.com/nglgzz/dots
  ./dots/fedora/config.sh
  ```

- Follow the dialogs.

**Notes**

It's possible to run the config script multiple times. Also, the dialogs allow
running a full initialization, or just a subset of it.
